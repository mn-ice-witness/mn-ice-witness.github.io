# Media Tweaks

Custom settings for media processing.

## OG Image Timestamps

Custom timestamps for OG image generation. Default is 2.0 seconds.

Format: `incident-slug: timestamp` (in seconds)

```
2026-01-13-aliya-rahman-car-window-hospitalized: 3.5
```

## High Quality Videos

Videos that should use less compression (CRF 23 instead of 35).
One slug per line.

```
2026-01-24-alex-pretti-shooting
2026-01-13-aliya-rahman-car-window-hospitalized
2026-01-07-teenager-tackled-snow
2026-01-08-target-richfield-employees
```
