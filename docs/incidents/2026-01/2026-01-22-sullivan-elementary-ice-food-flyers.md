The message read:

## What IS Verified

- **The school message exists**: Multiple sources confirm Anne Sullivan STEAM School sent this message via TalkingPoints on January 22, 2026
- **The message is authentic**: According to reporting, it was "confirmed by the school's administration"
- **The source**: Steve McPherson, a Minneapolis-based writer ([@steventurous.bsky.social](https://bsky.app/profile/steventurous.bsky.social)), shared the message on Bluesky, stating it came from the middle school's Signal chat

- Pretending to be local police
