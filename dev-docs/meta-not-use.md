# Meta Analysis Stories Not to Use

This file tracks meta analysis/investigative stories that were considered but not added to the About page's "Investigations & Analysis" section. Check this list before evaluating new stories.

## Criteria for Inclusion in About Page

Stories are added when they provide:
1. **In-depth investigative reporting** - Original research, data analysis, or document review
2. **Pattern documentation** - Systematic analysis of enforcement trends
3. **Minnesota-specific focus** - For Minnesota Coverage section
4. **National significance** - For National Investigations section (e.g., ProPublica investigations)

## Criteria for Exclusion

Stories are excluded when they:
1. **Duplicate existing coverage** - Same angle/data already covered by included story
2. **Breaking news only** - Daily news coverage without deeper analysis
3. **Opinion/editorial** - Commentary without original reporting
4. **Press releases** - Organization statements without investigative content
5. **Single incident focus** - Better suited as a source on the incident file itself
